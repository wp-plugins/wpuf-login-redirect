## Description ##

In Wp user fronted plugin user can post from fronted. Also user can edit his post from frontend. 

This plugin gives you the ability to redirect user of Wp user frontend (free/pro) to specific page. You can set specific page from the parent [WP User Frontend](https://WordPress.org/plugins/wp-user-frontend) plugin settings to redirect your user.

Author:  [@nayemDevs](https://twitter.com/nayemDevs)

### Features:  ###
1. Redirect user to specific page.
2. Admin can see the page list to select one of them to redirect 

== Installation ==
1. You have to install the parent plugin [WP User Frontend](https://WordPress.org/plugins/wp-user-frontend).
2. Please navigate to Wp admin->Plugins->Add new. Browse the plugin file and install it. 
3. After activating the plugin you will get option in WP User Frontend->Settings->General->Select redirect page.
